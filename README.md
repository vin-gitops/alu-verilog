# alu-verilog
This project implements ALU written in verilog along with it's testbench code
We can implement multiple arithmetic and logical operations using this ALU:
- ADD
- SUB
- AND
- OR
- XOR
- NOT A
- SHIFT RIGHT
- SHIFT LEFT

  Tools used:
  - EDA Playground

  Features:
  - testbench
  - Carry, Zero, and Overflow flags

Run Simulation Online:
https://www.edaplayground.com/x/uiK9
